About JS

JavaScript est un langage de programmation qui permet d'implémenter des mécanismes complexes sur une page web.
À chaque fois qu'une page web fait plus que simplement afficher du contenu statique, afficher du contenu mis à jour
à des temps déterminés, des cartes interactives, des animations 2D/3D, des menus vidéo défilants, ou autre,
JavaScript a de bonnes chances d'être impliqué.

JavaScript Workshop

Il s'agit de créer un site éducatif responsive proposant des exercices en JavaScript avec des solutions.
Votre application sera constituée d'un menu situé à gauche de votre écran (avec des sous-menus)
et un bloc principal divisé en trois sections: une pour l'énoncé, une deuxième pour simuler la solution
et une troisième pour écrire le script JS.
